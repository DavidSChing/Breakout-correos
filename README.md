# 📧 Correos Breakout - Plantillas HTML 2026

Sistema de gestión de plantillas de correos electrónicos para campañas de Breakout Capital y Zero to One.

## 🌐 Visualización Web

Puedes ver todas las plantillas renderizadas en: **[https://davidsching.github.io/Breakout-correos/](https://davidsching.github.io/Breakout-correos/)**

## 🎯 ¿Qué es esto?

Este es un repositorio centralizado que contiene todas las plantillas de correos HTML diseñadas para las campañas de comunicación de Breakout. Incluye correos para:
- **Breakout Capital**: Invitaciones a programa de aceleración y financiamiento
- **Zero to One**: Convocatorias para eventos y paneles
- **Otros**: Solicitudes de apoyo y comunicaciones institucionales

## 📋 Cómo usar las plantillas

### Para correos con extensiones de Gmail HTML:
Solo estas **3 versiones** tienen el formato correcto para copiar y pegar directamente en extensiones de Gmail:
- **Breakout Capital v1.3**
- **Zero to One v1.0**  
- **Zero to One v2.1**

**Pasos:**
1. Abre el archivo desde la página web
2. Haz clic en el botón "📋 Copiar código" (esquina superior derecha)
3. Pega el código directamente en tu extensión de Gmail HTML
4. Envía tu correo

### Para otros correos (visualización web):
Las demás versiones (v1.0, v1.1, v1.2, v2.0, v2.1) están optimizadas para verse en navegadores web pero también pueden copiarse.

## ⚠️ Diferencia entre versiones

- **Versiones con formato de email** (v1.3, Zero to One v1.0 y v2.1): Comienzan directamente con el contenido del correo, sin estructura HTML completa. Ideales para Gmail.
- **Versiones con formato web** (las demás): Tienen estructura HTML completa con `<!DOCTYPE html>`. Mejores para visualización pero pueden tener conflictos con algunas extensiones de Gmail.
